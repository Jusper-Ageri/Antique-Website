
Antique Creatives Limited — Static Website
=========================================

What's included
- index.html (single page)
- assets/css/styles.css
- assets/js/script.js
- assets/images/logo.jpg (your uploaded logo)

Local preview
-------------
Open index.html in a browser (double-click or `live server`).

Deploy to Netlify (drag & drop)
------------------------------
1. Go to https://app.netlify.com/drop
2. Drag the contents of the folder into the Netlify drop area (or upload the ZIP).
3. Configure a site name or connect a custom domain.

Deploy to GitHub Pages
----------------------
1. Create a new GitHub repository.
2. Commit the contents of this folder to the repository (branch: main).
3. In the repository go to Settings → Pages → Deploy from a branch → Choose 'main' and / (root).
4. Save. GitHub Pages will provide a URL (usually https://username.github.io/repo-name).

Notes
-----
- The form uses `mailto:` so submissions open the default email client.
- For a production form, integrate with Formspree, Netlify Forms, or a simple backend.
- Replace social links with your actual social URLs.
